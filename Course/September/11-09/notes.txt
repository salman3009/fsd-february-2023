Array methods:
//map,filter,reduce
map:
//Map function will not change original array.
//It will add/remove or check some condition and it will return new set of data
//It will consists of two parameters 
      (i)each iteration value 
      (ii)index value
Filter 
//Filter function will not change original array.
//It will return final result once it get boolean value true 
//It will consists of two parameters 
      (i)each iteration value 
      (ii)index value

Comparing map and filter:
map --> you can return new values
filter --> it will return same value
map --> 5 values --> 5 values
filter -->5values --> 2values

reduce 
//It will not change the original array 
//final result will return one single value
